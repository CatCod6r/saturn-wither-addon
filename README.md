# WitherBore
Break blocks with withers

## Prerequisites
1. Make sure you have enough egaps on you and in your hotbar
2. Enable AutoEat + AutoGap so you stay healthy :D
3. Have ElytraFly enabled.
4. Have both an empty hand and a sword in your hotbar
5. Acknowledge you will die AFKing for a while
6. Keybind the module so you can easily turn it off if the wither gets stuck on the roof

## How to use
Hit a wither (or multiple!) below 40% health, hover around height limit with ElytraFly in open air next to the sky logo you want to break, and enable the addon.
